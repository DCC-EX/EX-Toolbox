JMRI: Engine Driver Throttle for Android

Engine Driver version 0.4. This version adds a 2nd throttle control.

This application can connect to a JMRI WiThrottle server running on a PC, and in turn control your locomotives. Speed, direction, and up to 16 functions are supported. 

Before your Android device will allow you to install this program, you will need to permit the installation of non-Market applications. This is under Settings->Applications->Unknown sources. Then use your phone to download this file, and select it for installation.

Verify your Android can connect to your PC network via WiFi and get an address.

Make sure you are running one of the latest versions of JMRI (2.8 works, 2.9.6 works better) on your PC, connected to the layout.  Verify you can use the "Throttle" app to control a loco.

Start the WiThrottle function on JMRI.

On your phone, start the Engine Driver app, and shortly, your PC address should show up in "Discovered Services".  

Click on that address, or you can enter the address and port of your WiThrottle server directly.

On the next screen, enter the loco address, select short or long addressing, and press "Connect".

Operate your loco using the slider for speed, and the reverse, stop and forward buttons to control direction.

To release that loco, just press the Back button. This will also stop your loco (can be overridden in preferences).

Use the Function Buttons menu option to set the labels and function numbers to display. Blank labels will not display, so you can limit the buttons shown to those most useful to you. If you have set labels in a roster entry in JMRI, these labels can be copied from the dialog (access the menu after selecting a loco).
