EX-Toolbox for DCC-EX

Free, open source, app to help test and configure DCC-EX EX-CommandStations.

Changes for this version include:

  • Bug fix for the functions list on a roster entry

This free, open source, Android application connects to a DCC-EX EX-CommandStation using the native DCC-EX command protocol (only). Once connected, you can configure aspects of your EX-CommandStation including programming CVs.

Prerequisites
- DCC-EX EX-CommandStation:

EX-Toolbox can only connect directly to DCC-EX EX-CommandStations, that are in Station Mode or Access Point Mode, using the native DCC-EX command protocol. It cannot connect to JMRI or to a EX-CommandStation which is connected JMRI via USB.

Track Manager is only available when connected to EX-CommandStation version 4.2.7 and above.

Current features supported:

  • CV Programming in the programming track (Service Mode)
  • CV Programming in the main track (Operation Mode)
  • Servo motor testing and adjustment
  • Sensor testing
  • Current Meter (Only available when connected to EX-CommandStation version 4.2.20 and above.)
  • Track Manager (Only available when connected to EX-CommandStation version 4.2.7 and above.)

Operation:

On your phone, start the EX-Toolbox app and wait for your server's address to show up in "Discovered Services".

Click on the discovered address, or you can enter the address and port of your EX-CommandStation server directly.
There is an "Auto-Connect to First Server?" preference you can enable if desired.

Once connected the CV Programming screen will open. Use the drop down the select between Service Mode and Operation Mode. Swipe Left or right to get to the other screens.

Additional help is available on the DCC-EX website.

Development:

EX-Toolbox is free and open source software, part of the DCC-EX project. It is built in Android Studio.
It is based heavily on M.Steve Todd's Engine Driver app source code.
